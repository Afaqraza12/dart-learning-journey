
# Dart Learning Journey 🚀

Welcome to my repository tracking my progress as I learn the Dart programming language from scratch! This repository is a collection of hands-on practice tasks, exercises, and logical challenges designed to build a strong foundation in modern Dart.

## 🛠️ Concepts Covered So Far
* **Basics & Syntax:** Main function structure, semicolons, console printing.
* **Variables & Type System:** Statically typed variables (`String`, `int`, `double`), automatic type inference (`var`), and compile-time constants (`const`).
* **String Manipulation:** String interpolation (`$variable`) and text concatenation.
* **Control Flow:** Conditional operators (`if`, `else if`, `else`), logic gates (`&&`, `||`), and nested decision-making.
* **Modern Switch Statements:** Leveraging modern Dart 3.x pattern matching features (no-fallthrough cases).
* **Ternary Operators:** Clean, inline shorthand (`? :`) for conditional checks.

## 📁 Key Practice Programs In This Repo
* **Receipt Generator:** A cash register simulator calculating item prices, static tax rates, and total bills.
* **Login Validator:** A secure logic gate check using combined `&&` conditions.
* **Premium Discount & Cap System:** A complex, nested logic script evaluating membership tiers and dynamic discount thresholds with a maximum cap.
